18.La etiqueta "pre"

Despliegue para más información

La etiqueta "pre" sirve para presentar el texto exactamente como esta escrito en HTML.

Con pre:

                .center {
                    display: flex;
                    align-items: center;
                    justify-content: center;   
                }

Sin pre:

.center { display: flex; align-items: center; justify-content: center; }
